# Learning Hub

Learning Hub is a modern React-based application designed to track daily learning progress, manage weekly goals, and organize educational resources in a structured way.

---

## Overview

This project provides a clean and intuitive interface for tracking learning activities. It allows users to monitor progress, navigate through weekly plans, and access curated resources efficiently.

---

## Features

- Dashboard with overall progress tracking
- Week-wise structured learning system
- Daily task completion tracking using localStorage
- Organized resource management (links, videos, etc.)
- Smooth loading experience with skeleton UI
- Responsive and clean user interface

---

## Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- LocalStorage
